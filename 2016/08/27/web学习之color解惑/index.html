web学习之color解惑

发表于 2016-08-27   |   分类于 笔记   |  

问题

在web开发过程中，我经常会用到color这个属性，大多数时候也能找出自己想要的颜色，但心中对color的理解总是模模糊糊，不求甚解。这个周末总于抽出时间来好好归纳总结下，如有不对的地方，还请各位客官指正。

人眼中的颜色

我们生活在一个色彩缤纷的世界，但是，对于我们人眼看到的色彩只有两种：

  1. 一种是发光体发出的颜色，比如计算机显示器屏幕显示的颜色；
  2. 另一种是物体本身不发光，而是反射的光产生的颜色，比如看报纸和杂志上的颜色。

三原色

我们又知道任何颜色都是由三种最基本的颜色叠加形成的，这三种颜色我们称之为“三原色”

  1. 对于上面提到的第一种颜色，即发光体的颜色模式，又称之为“加色模式”，三原色是“红”、“绿”、“蓝”三种颜色。加色模式又称为“RGB模式”；
  2. 而对于印刷品这样的颜色模式，又称为“减色模式”，它的三原色是“青”、“洋红”、“黄”三种颜色。减色模式又称为“CMY模式”。

例如，在网页上要指定一种颜色，就要使用RGB模式来确定，方法是分别指定R/G/B，也就是红/绿/蓝三种原色的强度，通常规定，每一种颜色强度最低为0，最高为255，并通常都以16进制数值表示，那么255对应于十六进制就是FF，并把三个数值依次并列起来，以#开头。

网页中的三种表示形式

网页需要通过屏幕来展示，所以网页中的颜色采用的是RGB模式，下面介绍颜色值在网页中的四种表示形式：

  1. 直接写颜色的英文，例如：red（红）、blue（蓝）、green（绿）

  2. 用红色、绿色、蓝色的值设定color值，例如：rgb（204,213,9），（r代表red红色，g代表green绿色，b代表blue蓝色）

  3. 用红色、绿色、蓝色的值的百分比设定color值，例如：rgb（20%，20%，10%）。

  4. 最后一种就是我们上面提到的，使用颜色的十六进制值设定color值，这也是我们最常用的一种表示方法。#后面第一二位数字表示红色的值，三四位数字表示绿色的值，最后两位数字表示蓝色的值，例如：#c10a6b；c1表示红色的部分，0a表示绿色部分，6b表示蓝色部分。

色彩系统及其他

色彩系统，是一种常用的表示颜色的方式。应用于计算机屏幕，分别有：RGB色彩系统、CMYK色彩系统、YIQ色彩系统、YUV色彩系统、YCbCr色彩系统等。这里我们就不深入探讨了，如有兴趣可自行baidu google 。
